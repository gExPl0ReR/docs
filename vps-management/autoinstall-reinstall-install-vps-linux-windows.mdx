## Auto-Install your VPS with a fresh OS

This guide walks you through reinstalling the OS on your Virtual Private Server (VPS). Whether you’re starting over or switching operating systems, you’ll be able to set up your VPS with a clean install and a new password. **Make sure to back up your important files before proceeding, as reinstallation will wipe all existing data.**

## Turnkey Images

Turnkey images are pre-configured system snapshots that include an operating system and essential software, all set up and ready to be deployed immediately, akin to a product that's ready to use right out of the box. These images facilitate quick and easy setup for users who want to run servers or applications without going through the hassle of intense manual installation and configuration.

## How to Install a Turnkey Image

Download the .iso Image from https://www.turnkeylinux.org to your computer, and use our Cusom ISO function to upload the file to your server and have it installed. After the initial installation process of the .iso is completed, the Turnkey appliance will be deployed on your VPS. Finish the installation in a VNC session.

After activating VPS with Windows, one of the first tasks is often to install a web browser like Google Chrome. This guide provides a detailed, step-by-step process for installing Google Chrome on your rented Windows Server VPS, ensuring you have a secure and efficient browsing experience right from the start.

## How to get going&#x20;

1. [Order a VPS](https://www.edisglobal.com/) with pre-installed Windows Server OS.

2. [Connect to the server via RDP](../getting-started/windows-remote-desktop), port 3389.

3. Open the **Server Manager**. The Server Manager usually opens automatically when the operating system starts. However, if the Server Manager does not open automatically, click on the **Start** button. Then, find and select "**Server Manager**" in the **Start** menu. It is typically pinned by default when you install Windows Server. ![](/assets/y9B0AGVAxYeSyeZVyaDis-7QVfnkzj8whiU7FU1x-GV-20240925-111932.png)

4. Choose the **Local Server** section on the Server Manager![](/assets/y9B0AGVAxYeSyeZVyaDis-6mlwvxUHmMw4B2Uhjy3fx-20240925-112129.png)

5. Disable the IE Enhanced Security Configuration setting by clicking on "On":![](/assets/y9B0AGVAxYeSyeZVyaDis-ZadLaSIVTwxMBR9WJxb-X-20240925-112756.png)

6. In the new window, disable the security configuration for both Administrators and Users, then click "Apply."![](/assets/y9B0AGVAxYeSyeZVyaDis-BuKWLMA8OqXyiMFd0ZU2a-20240925-112928.png)

7. Open the IE browser. Enter the link: [http://www.google.com/chrome](http://www.google.com/chrome) to the address bar and follow to download the Google Chrome browser. When the page opens click on **Download Chrome** button.![](/assets/y9B0AGVAxYeSyeZVyaDis-O_vkRcwSj_BcZROqvwYxS-20240925-113016.png)

8. Click to **Run** button on the new pop-up window that will located in the bottom section of the screen.![](/assets/y9B0AGVAxYeSyeZVyaDis-T6_8Y4a5mP3LvYEuxojJp-20240925-113951.png)

9. Set Google Chrome as the default browser.![](/assets/y9B0AGVAxYeSyeZVyaDis-CO7-zXgNq_r2mTA79IpJb-20240925-114316.png)

10. Congratulations! Google Chrome has been successfully installed on your Windows Server. This installation enhances the server's capabilities by providing a fast and secure web browser.![](/assets/y9B0AGVAxYeSyeZVyaDis-JSmf__3EKINSsSxUUMSsU-20240925-114409.png)

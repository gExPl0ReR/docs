After activating VPS with Windows, one of the first tasks is often to install a web browser like Google Chrome. This guide provides a detailed, step-by-step process for installing Google Chrome on your rented Windows Server VPS, ensuring you have a secure and efficient browsing experience right from the start.

## How to get going

<Steps>
  <Step title="Order a Windows VPS">
    [Order a VPS](https://www.edisglobal.com/) with pre-installed Windows Server OS.
  </Step>

  <Step title="Connect via RDP">
    [Connect to the server via RDP](../getting-started/windows-remote-desktop) on port 3389.
  </Step>

  <Step title="Open Server Manager">
    The **Server Manager** usually opens automatically when the operating system starts. If it doesn't, click the **Start** button and select **Server Manager** from the Start menu — it's typically pinned by default.

    ![Opening Server Manager on Windows Server](/assets/y9B0AGVAxYeSyeZVyaDis-7QVfnkzj8whiU7FU1x-GV-20240925-111932.png)
  </Step>

  <Step title="Select Local Server">
    Choose the **Local Server** section in Server Manager.

    ![Local Server section in Server Manager](/assets/y9B0AGVAxYeSyeZVyaDis-6mlwvxUHmMw4B2Uhjy3fx-20240925-112129.png)
  </Step>

  <Step title="Open IE Enhanced Security settings">
    Disable the IE Enhanced Security Configuration setting by clicking **On**.

    ![IE Enhanced Security Configuration toggle](/assets/y9B0AGVAxYeSyeZVyaDis-ZadLaSIVTwxMBR9WJxb-X-20240925-112756.png)
  </Step>

  <Step title="Disable IE Enhanced Security">
    In the new window, disable the security configuration for both **Administrators** and **Users**, then click **Apply**.

    ![Disabling IE Enhanced Security for Administrators and Users](/assets/y9B0AGVAxYeSyeZVyaDis-BuKWLMA8OqXyiMFd0ZU2a-20240925-112928.png)
  </Step>

  <Step title="Download Chrome">
    Open Internet Explorer and enter [http://www.google.com/chrome](http://www.google.com/chrome) in the address bar. When the page opens, click **Download Chrome**.

    ![Downloading Chrome from google.com/chrome](/assets/y9B0AGVAxYeSyeZVyaDis-O_vkRcwSj_BcZROqvwYxS-20240925-113016.png)
  </Step>

  <Step title="Run the installer">
    Click the **Run** button in the pop-up at the bottom of the screen.

    ![Running the Chrome installer](/assets/y9B0AGVAxYeSyeZVyaDis-T6_8Y4a5mP3LvYEuxojJp-20240925-113951.png)
  </Step>

  <Step title="Set Chrome as the default browser">
    Set Google Chrome as the default browser.

    ![Setting Chrome as the default browser](/assets/y9B0AGVAxYeSyeZVyaDis-CO7-zXgNq_r2mTA79IpJb-20240925-114316.png)
  </Step>

  <Step title="Finish">
    Congratulations! Google Chrome has been successfully installed on your Windows Server, providing a fast and secure web browser.

    ![Chrome successfully installed on Windows Server](/assets/y9B0AGVAxYeSyeZVyaDis-JSmf__3EKINSsSxUUMSsU-20240925-114409.png)
  </Step>
</Steps>

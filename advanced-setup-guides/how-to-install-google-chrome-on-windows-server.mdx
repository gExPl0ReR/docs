After activating VPS with Windows, one of the first tasks is often to install a web browser like Google Chrome. This guide provides a detailed, step-by-step process for installing Google Chrome on your rented Windows Server VPS, ensuring you have a secure and efficient browsing experience right from the start.

## How to get going 

1. [Order a VPS](https://www.edisglobal.com/) with pre-installed Windows Server OS.
2. [Connect to the server via RDP](../getting-started/windows-remote-desktop), port 3389.
3. Open the **Server Manager**. The Server Manager usually opens automatically when the operating system starts. However, if the Server Manager does not open automatically, click on the **Start** button. Then, find and select "**Server Manager**" in the **Start** menu. It is typically pinned by default when you install Windows Server. <img src="/assets/y9B0AGVAxYeSyeZVyaDis-7QVfnkzj8whiU7FU1x-GV-20240925-111932.png" alt="Install Chrome on Windows Server" />
4. Choose the **Local Server** section on the Server Manager <img src="/assets/y9B0AGVAxYeSyeZVyaDis-6mlwvxUHmMw4B2Uhjy3fx-20240925-112129.png" alt="Install Chrome on Windows Server" />
5. Disable the IE Enhanced Security Configuration setting by clicking on "On": <img src="/assets/y9B0AGVAxYeSyeZVyaDis-ZadLaSIVTwxMBR9WJxb-X-20240925-112756.png" alt="Install Chrome on Windows Server" />
6. Click to **Run** button on the new pop-up window that will be located in the bottom section of the screen. <img src="/assets/y9B0AGVAxYeSyeZVyaDis-T6_8Y4a5mP3LvYEuxojJp-20240925-113951.png" alt="Install Chrome on Windows Server" />
7. Set Google Chrome as the default browser. <img src="/assets/y9B0AGVAxYeSyeZVyaDis-CO7-zXgNq_r2mTA79IpJb-20240925-114316.png" alt="Install Chrome on Windows Server" />
8. Congratulations! Google Chrome has been successfully installed on your Windows Server. This installation enhances the server's capabilities by providing a fast and secure web browser. <img src="/assets/y9B0AGVAxYeSyeZVyaDis-JSmf__3EKINSsSxUUMSsU-20240925-114409.png" alt="Install Chrome on Windows Server" />

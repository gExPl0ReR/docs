Converts Title Case headings across all MDX documents to sentence case, following Mintlify's writing convention.

**Scope**
- 129 files updated, 481 headings changed
- Only headings detected as Title Case were converted (≥70% of words capitalized)
- Frontmatter, code fences, and code spans were left untouched
